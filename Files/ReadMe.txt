Make sure you have:
	JDK installed (prefferably the latest version) (when writing in cmd "java --version", it should work)
	JavaFX library on your computer
	Will to live

How to start:
	Open a Terminal in this directory (Start command line, CD into this directory)
        Run the following command in your terminal

java --module-path ".\javafx-sdk-17.0.1\lib" --add-modules javafx.controls,javafx.fxml -jar .\Quizz.jar

Good luck and enjoy
